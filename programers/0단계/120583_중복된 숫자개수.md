# 문제

중복된 숫자 개수
https://school.programmers.co.kr/learn/courses/30/lessons/120583

## 1. 원하는 것

정수 배열에 포함된 n이 몇개인지

## 2. 아는 것

정수배열array와 정수n

## 3. 계획

array에서 n을 찾아 갯수를 반환한다

## 4. 풀이

```
  function solution(array, n) {
    const answer = array.filter((item) => item === n).length;

    return answer;
}
```
